4.4 Löschen

Tast.MenüKommando
d‘Delete markup at point’standoff-markup-delete-range-at-point

Zum Löschen eines Markup-Elements ist der Point auf dieses Element zu positionieren. Es muss zudem hervorgehoben sein (highlight), und es muss sich um das einzige hervorgehobene Element an der Position von Point handeln, überlappende andere Elemente sind also auszublenden. Wird nicht genau ein hervorgehobenes Element an Point gefunden, erfolgt eine Fehlermeldung.
